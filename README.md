# Quran Web App
Full-stack Quran reader — Next.js 14 SSG frontend + Hono/Node backend.

## Run
1. Backend:  cd backend && cp .env.example .env && npm install && npm run dev
2. Frontend: cd frontend && cp .env.example .env.local && npm install && npm run dev
3. Open http://localhost:3000

Data source: https://api.alquran.cloud (quran-uthmani + en.sahih)
